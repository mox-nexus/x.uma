pub fn register_core_matchers<Ctx: 'static>(
    builder: RegistryBuilder<Ctx>,
) -> RegistryBuilder<Ctx>
Expand description

Register core built-in matchers (BoolMatcher, StringMatcher).

Call this in domain register() functions to avoid duplicating core matcher registrations. Domain-specific inputs are then added on top.

§Example

ⓘ
pub fn register(builder: RegistryBuilder<MyCtx>) -> RegistryBuilder<MyCtx> {
    rumi::register_core_matchers(builder)
        .input::<MyInput>("xuma.my.v1.MyInput")
}
