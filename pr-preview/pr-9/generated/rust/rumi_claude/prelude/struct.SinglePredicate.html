Struct SinglePredicate

Expand description

A single predicate: combines a DataInput with an InputMatcher.

This is where domain-specific (DataInput) meets domain-agnostic (InputMatcher). The SinglePredicate extracts data from the context using the input, then passes it to the matcher for evaluation.

§INV (Dijkstra): None → false

If the DataInput returns MatchingData::None, the predicate evaluates to false. This is a critical invariant that simplifies reasoning about matcher behavior.

§Example

ⓘ
let predicate = SinglePredicate::new(
    Box::new(HeaderInput::new("content-type")),
    Box::new(ContainsMatcher::new("json")),
);
let result = predicate.evaluate(&request);
